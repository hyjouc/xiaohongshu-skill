# Xiaohongshu Post Template

## Title Patterns

- 我试了 X，终于把 Y 讲明白了
- 新手做 X，先看完这几点
- X 不是不会做，是少了这个流程
- 适合收藏的 X 清单
- 用 X 做 Y，我建议从这 5 步开始

## Body Structure

开头：两行内完成一个真人观察 + 具体场景。

正文：

背景 / 为什么重要
短解释，不写长段。

方法 / 流程
每段 1-2 行。

具体步骤
用短句和符号分隔。

✅ 适合谁
- 人群 1
- 人群 2

⚠️ 注意事项
一句边界或提醒。

结尾：轻互动，不强迫。

## Anti-AI Voice

- 不写“随着时代发展 / 总的来说 / 综上所述 / 核心在于 / 赋能 / 闭环 / 抓手”。
- 不写得像产品说明书或培训稿。
- 不要每段都同样长度，也不要机械 01-06 排列。
- 多写具体场景：用户输入了什么、哪里卡住、改完以后少了什么返工。
- 可以有轻微口语感，但不要装熟、不要夸张。
- 最后检查一遍，删掉空话、万能总结和太整齐的并列句。
